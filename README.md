# Teslo API

1. Clonar el proyecto
2. Instalar dependencias
```yarn install```
3. Clonar el archivo ```.env.template``` y renombrar por ```.env```
4. Cambiar las variables de entorno
5. Levantar la base de datos (Docker)
```docker-compose up -d```
6. Levantar el modo de desarrollo
```yarn run start:dev```
